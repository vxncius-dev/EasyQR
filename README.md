# EaseQR

EaseQR is a high-performance QR code generator designed for rapid sharing of links, text, and local files. It focuses on speed, simplicity, and a clean workflow.

## Core Capabilities

- **Instant Generation**: Paste links or text to create a QR code on the fly.
- **Local File Support**: Drag-and-drop files up to 100MB to generate a QR code for a local blob URL.
- **Intuitive Workflow**: Supports drag-and-drop file processing and clipboard interaction.
- **History List**: Keeps recent items in a bottom-sheet list for quick reuse.

## Technical Details

Built with a focus on speed and reliability using:

- **ZXing**: For robust QR code generation.
- **Vanilla JS**: No heavy frameworks, ensuring fast load times and a responsive experience.

## Usage

1. Open the application.
2. Enter text, paste a URL, or drag a file onto the interface.
3. The QR code updates automatically.
4. Scan or share the generated content instantly.
