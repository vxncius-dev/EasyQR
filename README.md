# EaseQR

EaseQR is a high-performance QR code generator featuring integrated temporary file hosting. It is designed for rapid sharing of links, text, and documents with automatic expiration.

[![EaseQR Preview Placeholder](https://via.placeholder.com/800x450?text=Insert+Site+Screenshot+Here)](file:///d:/Vini/HTML/EaseQR-main/EaseQR-main/screenshot.png)

## Core Capabilities

- **Instant Generation**: Paste links or text to create a QR code on the fly.
- **Temporary Hosting**: Upload files up to 100MB to receive a direct download link and a corresponding QR code.
- **Intuitive Workflow**: Supports drag-and-drop file processing and clipboard interaction.
- **Privacy Focused**: Files are stored temporarily on Tmpfiles.org and expire automatically.

## Technical Details

Built with a focus on speed and reliability using:

- **ZXing**: For robust QR code generation.
- **Tmpfiles.org API**: For ephemeral file storage.
- **Vanilla JS**: No heavy frameworks, ensuring fast load times and a responsive experience.

## Usage

1. Open the application.
2. Enter text, paste a URL, or drag a file onto the interface.
3. The QR code updates automatically.
4. Scan or share the generated content instantly.
